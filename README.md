# Weather Analysis 

Using Openweather API, I gathered data of over 500 cities and find out which is the best place to take a vacation in. The data analyzed for each hemisphere was

* Cloudiness
* Humidity
* Temperature
* Wind speed



## Built with
* Python 
* Pandas
* Matplotlib
